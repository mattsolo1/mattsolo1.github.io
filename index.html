matthew solomonson

biologist/developer

I build programs for exploring large biological datasets on the web.

Biological data is exploding in size and complexity. To make sense of it, we need to engineer a new generation of interactive visualization tools that are accessible to researchers, clinicians, hobbyists, and people without scientific training.

I enjoy learning, designing, creating, and being a part of significant collaborative initiatives. I am excited to work with other researchers to create translatable interfaces that accurately communicate complex information while getting people excited about biology.

Technologies I use

I am interested in next-generation web paradigms that simplify application development and maintainability. I use JavaScript, the React view library for manipulating the DOM, and the Redux pattern for managing application state. Where possible, I compose simple/reusable/modular web components from scratch through behavior-driven development, writing unit tests as I go. I can work with NoSQL databases and build simple restful APIs. I can deploy dockerized web applications on cloud computing platforms like Amazon Web Services.

  • Language/environment: Functional ES6 JavaScript, node.js
  • Data processing, visualization: HTML/JSX, PostCSS, SVG, Ramda, D3
  • Module bundling/compilation: Webpack, Babel
  • DOM manipulation/application state: React, Redux
  • Unit testing: Mocha
  • Databases: MongoDB, RethinkDB
  • Infrastructure: Unix/Linux, git, Docker, Amazon Web Services
  • Data analysis: Python, IPython, matplotlib, pandas

Projects

exome browser

This web application explores the dataset released by the Exome Aggregation Consortium (ExAC), an aggregated analysis of protein-coding sequences from over 60,000 humans.

Exomes are the protein coding regions of the genome and important factors in disease and human traits. I am not affiliated with the ExAC consortium. The browser is a UI concept and was conceived as a way for me to learn about software development and human genomics. Only tested on latest versions of Chrome/Firefox so far. hmmerclust

hmmerclust is a Python package for detecting gene clusters across thousands of bacterial genomes for comparative molecular systems analysis in an IPython notebook.

Source

coot control

For wirelessly refining protein structures with an iPad. The app provides a tactile control surface with tailor-made joysticks, buttons, and sliders to make tedious molecular building tasks fluid and ergonomic. It was written in Objective-C and communicates to a Python server running on the computer.
