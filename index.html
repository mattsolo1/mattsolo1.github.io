I build programs for exploring large biological datasets on the web.

Biological data is exploding in size and complexity. To make sense of it, we need to engineer a new generation of interactive visualization tools that are accessible to researchers, clinicians, hobbyists, and people without scientific training.

I enjoy learning, designing, creating, and being a part of significant collaborative initiatives. I am excited to work with other researchers to create translatable interfaces that accurately communicate complex information while getting people excited about biology.

Technologies I use

Below are some of the technologies I am familiar with. I am interested in next-generation web paradigms that simplify application development and maintainability. I use JavaScript, the React view library, and the Redux pattern to maintain application state. These technologies are powerful for displaying data when combined with Ramda.js, D3, CSS, and SVG. Where possible, I like to compose simple/reusable/modular web components from scratch through behavior-driven development, writing unit tests as I go. I can work with NoSQL databases and build simple restful APIs. I can deploy dockerized web applications on cloud computing platforms like Amazon Web Services.

  • Language/environment: Functional ES6 JavaScript, node.js
  • Data processing, visualization: HTML/JSX, PostCSS, SVG, Ramda, D3
  • Module bundling/compilation: Webpack, Babel
  • DOM manipulation/application state: React, Redux
  • Unit testing: Mocha
  • Databases: MongoDB, RethinkDB
  • Infrastructure: Unix/Linux, git, Docker, Amazon Web Services
  • Data analysis: Python, IPython, matplotlib, pandas

Source

This application can be used to explore the dataset released by the Exome Aggregation Consortium (ExAC), which describes protein-coding sequences from 60,000 individuals. I am not affiliated with ExAC and have used their dataset to learn about software development and personal genomics.

hmmerclust

Source

hmmerclust is a Python package for detecting gene clusters across thousands of bacterial genomes for comparative molecular systems analysis in an IPython notebook.
